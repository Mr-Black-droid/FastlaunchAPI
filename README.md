FastLaunch API: SpaceX Mission Control
FastLaunch API is a cloud-ready, commercial orbital launch manifest service. Built with FastAPI and SQLite, it provides a robust backend for managing rocket products and customer launch orders with built-in business logic for payload discounts.

Project Overview
This project was developed for the Introduction to Cloud Technology course to demonstrate a "Mature API" architecture. It features:

Full CRUD Operations: Create, Read, and Delete launch missions.

Business Logic: Automatic 10% price reduction for rideshare missions.

Persistence: All orders are stored in a local SQLite database.

Security: Protected endpoints using HTTP Basic Auth with credentials stored in environment variables.

Interactive Documentation: Automated Swagger UI and ReDoc for easy API testing.

Tech Stack
Language: Python 3.10+

Framework: FastAPI

Database: SQLite3

Frontend: HTML5, CSS3 (Galaxy-themed), JavaScript (Starfield Canvas API)
